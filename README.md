### To run this website locally:

1. Clone this repository:<br>

   ```bash

   ```

   ```bash
   cd forever-uselesss.github.io
   ```

2. Start python server: <br>
   Linux:
   ```bash

   ```
   Windows:
   ```bash
   python start_server.py
   ```
3. Open Browser and type in:
   ```bash
   localhost:8080
   ```
